Ayushman Bharat Health Data Intelligence Repository (ABHDIR)

A structured, analytics-ready health and epidemiological data repository designed to support Ayushman Bharat and PMJAY initiatives through predictive modeling, disease surveillance, and longitudinal beneficiary wellness analysis.
